# Frontend Developer Portfolio

A clean personal portfolio built with **HTML, CSS, and vanilla JavaScript**.
Lightweight, responsive, and designed to showcase projects, skills, and contact information.


## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript

No frameworks or build tools required.


## Project Structure

portfolio/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
└── assets/ (just the svg favicon for now)


## Features

* Responsive layout
* Smooth scrolling and reveal animations
* Interactive UI elements
* Minimal and performance-focused design
* Easy to customize


## Deployment

The site can be deployed easily on platforms like:

* Netlify
* Vercel
* GitHub Pages

Simply upload or connect the repository and deploy.
